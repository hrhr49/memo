# [Hugo](https://github.com/gohugoio/hugo)とは

* 静的サイトジェネレータと呼ばれるソフトウェアの一種
* Go言語で作られているため、ビルドが高速らしい


## インストール

homebrewでインストールできる。

```sh
brew install hugo
```
